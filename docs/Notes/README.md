# Working Notes

Notes record observations, investigations, validation results, and design
questions while they are still being explored. They are evidence, not
automatically accepted architecture.

Notes commonly graduate into a plan, an Architectural Review, an ADR, or the
archive. When a note becomes obsolete, preserve it as history instead of
silently rewriting its conclusions.
